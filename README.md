# Yatri Mitra Shared Mobility App

A modern Android app for shared mobility services. It helps users find rides, share trips, manage bookings, and track travel with a fast and user friendly system designed for local transport solutions.

## Features

*   **Ride Finding**: Quickly find available rides in your area.
*   **Trip Sharing**: Easily share your own trips with others.
*   **Booking Management**: Manage all your shared mobility bookings in one place.
*   **Real-time Tracking**: Track your travel in real-time for safety and convenience.
*   **User-Friendly Interface**: Designed for a smooth and intuitive user experience.
*   **Local Focus**: Optimized for local transport solutions.
